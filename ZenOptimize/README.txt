What is ZenOptimize?
A lightweight tool that applies safe Windows registry tweaks to improve system responsiveness, reduce latency, and enhance gaming performance.

v1.1 NEW FEATURES:
✓ Windows Game Mode enabled
✓ Hardware-Accelerated GPU Scheduling (HAGS) with auto-detection
✓ Focus Assist & notification suppression
✓ Xbox services optimization (controller support preserved)
✓ Comprehensive logging system
---------------------------------------------------------------------------------------------
How to Use

Run ZenOptimize_v1.1.exe as Administrator

Choose:

1 → Apply Optimizations

2 → Restore Previous Settings

3 → Exit

A backup is created automatically before any changes.
If HAGS is enabled, you'll be prompted to restart (recommended).
---------------------------------------------------------------------------------------------
Safety

- Only modifies specific registry keys

- Always creates a backup

- Fully reversible

Recommended: Create a Windows Restore Point before using any optimization tool
