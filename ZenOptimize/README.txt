ZenOptimize – Windows Performance Optimizer
by Sid FPS
---------------------------------------------------------------------------------------------
What is ZenOptimize?
A lightweight tool that applies safe Windows registry tweaks to improve system responsiveness, reduce latency, and enhance gaming performance. No bloat, no background services.
---------------------------------------------------------------------------------------------
How to Use

Extract the ZIP

Run ZenOptimize.exe as Administrator

Choose:

1 → Apply Optimizations

2 → Restore Backup

3 → Exit

A backup is created automatically before any changes.
---------------------------------------------------------------------------------------------
Safety

- Only modifies specific registry keys

- Always creates a backup

- Fully reversible

Recommended: Create a Windows Restore Point before using any optimization tool
